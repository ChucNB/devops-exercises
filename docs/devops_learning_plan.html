Lộ trình học DevOps trong 8 tuần

Tổng quan

Thời lượng 8 tuần (4 ngày học/tuần + 1 buổi ôn tập nhẹ, mỗi buổi ~120 phút)

Phù hợp Backend Java Developer muốn chuyển sang DevOps/SRE

Tài nguyên Repository devops-exercises và dự án Java cá nhân

Phương pháp Mỗi ngày 1 chủ đề chính + thực hành/ghi chú trong khung 2 giờ buổi tối

Mục tiêu trọng tâm

Nền tảng vận hành

Làm chủ Linux, Git, networking, monitoring cơ bản.

Tự động hoá & Container

Triển khai Docker, CI/CD, scripting hỗ trợ dịch vụ Java.

Hạ tầng & Cloud

Hiểu AWS core services, Terraform và pipeline IaC.

Độ tin cậy hệ thống

Quan sát, quản lý sự cố, chuẩn bị phỏng vấn DevOps/SRE.

Mẹo & tài nguyên bổ sung

  • Song song hoàn thiện ghi chú cá nhân bằng Notion/Obsidian để dễ review.
  • Dùng dự án Java của bạn làm “sân chơi” áp dụng từng chủ đề.
  • Chia sẻ tiến độ mỗi tuần với mentor hoặc đồng đội để nhận phản hồi.
  • Khi thiếu thời gian, ưu tiên chủ đề trực tiếp hỗ trợ pipeline hiện tại.
  • Có thể ghép học đôi với đồng nghiệp để luyện giải bài tập khó.
  • Tổng kết mỗi tuần bằng blog nội bộ hoặc retro nhỏ cho bản thân.
